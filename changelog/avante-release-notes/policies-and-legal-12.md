# Avante 4.0.0

#### **Released April, 2024** <a href="#released-april-9-2026" id="released-april-9-2026"></a>

### Added

* **Pre-order template:** Switch a product to pre-order mode by assigning the pre-order template in Shopify Admin → Products → Theme template. The Add to Cart button automatically changes to "Pre-order". See Product page guide.
* **Combined Listings support:** Products using Shopify's Combined Listings feature are now fully supported.
* **Linked products on product cards:** Show color variant products as swatches on collection cards. See Variant images vs. Color Swatches guide.
* **New sections:** New Arrivals Collection, Featured Collection with Product Grid, New Arrivals Tabs.
