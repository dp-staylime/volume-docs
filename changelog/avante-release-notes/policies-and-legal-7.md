# Avante 7.0.0

#### **Released January, 2025** <a href="#released-april-9-2026" id="released-april-9-2026"></a>

### Added

* **Events section:** Showcase upcoming events with ticket CTAs and calendar add options. Add from the sections list in the Theme Editor.
* **Mega menu drawer:** A full-screen off-canvas menu layout. Enable in Theme settings → Navigation → Menu drawer style → Mega menu drawer.
* **Countdown timer on product page:** Add urgency to time-sensitive promotions. Add a Countdown timer block in the Product overview section.
* **Video autoplay on product page:** Videos can now auto-play when in view. Enable per product media in the Product overview section.
