# Avante 13.0.0

#### **Released March, 2026** <a href="#released-april-9-2026" id="released-april-9-2026"></a>

### Added

* **Fairmont preset:** New preset designed for home and interior brands — editorial-style, polished out of the box.
* **Disable layout margins (edge-to-edge):** A new _Disable layout margins_ setting lets images and sections extend to the full screen width. Find it in [Theme settings → Appearance](../../theme-settings/appearance.md).
* **Section categories in Theme Editor:** Sections in the Theme Editor are now grouped by category, making it easier to find what you need.
* **Cascade animations:** Cascade animations added to Menu drawer, Cart drawer, News drawer, and Store selector drawer. Enable per-drawer in [Theme settings → Animation](../../theme-settings/animation.md).
* **Compare-at price color:** Customize the color of strikethrough prices. _Theme settings → Colors → Custom compare-at price color._
* **Smooth anchor link scrolling:** Clicking anchor links now scrolls smoothly. Enable in _Theme settings → Animation → Enable anchor links smooth scroll._
* **Availability toggle in filters:** A new filter option lets customers filter by "In stock" availability. Configure via the [Search & Discovery app](../../collections-and-products/filters-and-search.md).
