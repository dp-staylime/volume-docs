# Avante 11.0.0

#### **Released August, 2025** <a href="#released-april-9-2026" id="released-april-9-2026"></a>

### Added

* **Panorama preset:** New preset. Install from the Shopify Theme Store for the full demo.
* **Custom typography preset:** Full independent control over every text style. Select Custom in Theme settings → Typography.
* **Breadcrumbs:** Enable in Theme settings → Breadcrumbs. Improves navigation and SEO on collection and product pages.
* **Color blending for product cards:** Applies a subtle tint to light product image backgrounds for a cleaner grid. Enable in Theme settings → Product card → Color blending.
* **Store Selector in menu drawer:** The Store Selector can now be added as a block inside the menu drawer.
