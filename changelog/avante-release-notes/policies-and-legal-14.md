# Avante 1.4.0

#### **Released December, 2023** <a href="#released-april-9-2026" id="released-april-9-2026"></a>

### Added

* **Popup section:** Add a popup to any page via the Theme Editor. Supports text, images, and newsletter signup.
* **Age verification section:** A built-in age gate for stores selling age-restricted products.
* **EU translations:** French, German, Spanish, and Italian translations now included out of the box.
