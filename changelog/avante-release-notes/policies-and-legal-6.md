# Avante 8.0.0

#### **Released March, 2025** <a href="#released-april-9-2026" id="released-april-9-2026"></a>

### Added

* **Transparent mobile header:** The transparent header now works on mobile too. Enable in Theme settings → Appearance → Transparent header and sidebars → Enable on mobile. See Transparent header guide.
* **Up to 2048 variants:** Avante now supports up to 2,048 variants per product. Works automatically — no configuration needed.
* **Product card sliders on hover:** Product cards can show a slider when hovered. Enable per section: On hover → Show: Slider.
* **Full RTL support:** Right-to-left language layouts are now supported across all sections.
* **Image lazy loading effects:** New preloader animations for images. Enable in Theme settings → Animation → Images → Lazy loading.
