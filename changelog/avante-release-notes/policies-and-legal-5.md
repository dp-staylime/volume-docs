# Avante 9.0.0

#### **Released May, 2025** <a href="#released-april-9-2026" id="released-april-9-2026"></a>

### Added

* **Store Selector:** Let customers switch between regional stores from the header and product cards. See Store Selector setup guide.
* **Preload links on hover:** Pages start loading when a customer hovers over a link — navigation feels nearly instant. Enable in Theme settings → Animation → Preload links on hover.
* **Page fade-in effect:** A smooth fade transition on page load. Enable in Theme settings → Animation → Enable fade-in on page load.
