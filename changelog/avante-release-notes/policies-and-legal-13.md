# Avante 3.0.0

#### **Released March, 2024** <a href="#released-april-9-2026" id="released-april-9-2026"></a>

### Added

* **Linked products:** Connect separate products by color and display them as swatches on the product page and collection cards. See Link up products guide.
* **Horizontal filter layout:** A new filter style option for stores using sidebar-based navigation. Configure in the collection template settings.
* **Terms and Conditions checkbox:** Add a T\&C checkbox to cart and contact forms via a block in the Cart section.
