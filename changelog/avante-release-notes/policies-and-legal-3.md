# Avante 12.0.0

#### **Released October, 2025** <a href="#released-april-9-2026" id="released-april-9-2026"></a>

### Added

* **Adelle preset:** New preset. Install from the Shopify Theme Store for the full demo layout.
* **News Drawer:** A full-screen drawer section for articles, announcements, or editorial content.
* **Shoppable Media Slider:** A new section combining media with clickable product tags.
* **Stories Slideshow:** A vertical story-style slideshow section.
* **Horizontal Scrolling Banners:** A banner strip with horizontal scroll on mobile.
* **Discount code in Cart drawer:** Customers can enter discount codes directly in the cart drawer. Enable in Theme settings → Cart drawer.
