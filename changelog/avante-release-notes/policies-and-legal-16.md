# Avante 1.1.0

#### **Released November, 2023** <a href="#released-april-9-2026" id="released-april-9-2026"></a>

### Added

* **Filter layouts:** Choose between horizontal, vertical, or drawer filter styles on collection and search pages. Configure in the collection template section settings.
* **Infinite scroll / Load more:** Replace traditional pagination with infinite scroll or a Load more button. Set per collection template.
