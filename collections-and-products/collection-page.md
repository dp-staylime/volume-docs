# Collection page

Configure your collection pages in **Theme Editor → Collections → Default collection** (or a custom collection template).

## Product grid settings

| Setting                     | Notes                                                                                     |
| --------------------------- | ----------------------------------------------------------------------------------------- |
| Products per row            | 2–4 columns on desktop                                                                    |
| Infinite scroll / Load more | Choose how additional products load                                                       |
| Image ratio                 | Keep consistent across products for a clean grid                                          |
| Show short description      | Requires [Product subtitle](../features-and-advanced-setup/product-subtitle.md) metafield |
| Enable color pills          | Shows color swatches on product cards                                                     |

## Breadcrumbs

Enable in **Theme settings → Breadcrumbs** (v11). Breadcrumbs improve navigation and SEO by showing the path: Home → Collection → Product.

## Sorting and filtering

Avante supports Shopify's native filters via the **Search & Discovery** app (free, by Shopify):

1. Install [Search & Discovery](https://apps.shopify.com/search-and-discovery) from the App Store
2. Configure filters in the app (by availability, price, vendor, product type, or metafields)
3. Enable **Filter swatches** in **Theme settings → Filter swatches** for visual color filters

> Filters only appear on collection pages that have products matching the configured filter criteria.
