# Product page

Configure your product pages in **Theme Editor → Products → Default product** (or a custom template).

## Gallery

| Setting        | Notes                                      |
| -------------- | ------------------------------------------ |
| Gallery style  | Stacked, Grid, Slideshow, or Thumbnails    |
| Gallery size   | Narrow, Medium, Wide, Full                 |
| Zoom           | Enable image zoom on hover/click           |
| Video autoplay | Auto-play product videos when in view (v7) |

## Product overview

The **Product overview** section contains the core purchase elements. Key blocks:

* **Title, Price, Variants** — always present
* **Add to cart** — main purchase button
* **Sticky add to cart** — button stays visible while scrolling (v5.2). Enable in Product overview section settings.
* **Collapsible tabs** — for size guides, materials, shipping info
* **Ask a Question** — built-in contact form, no app required (v5.1)

## Pre-order products

Switch a product to pre-order in **Shopify Admin → Products → \[product] → Theme template** — change from `Default product` to `pre-order`. The Add to Cart button automatically changes to "Pre-order".

## Recommended products

Add a **Product recommendations** section below the Product overview to show related items powered by Shopify's recommendation engine.
