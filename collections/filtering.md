# Filtering

Configure filters on collection pages from **Theme Editor → Collection page → Product Grid section**.

`[SCREENSHOT: Filtering panel on collection page]`

---

## Filter layout styles

Three layout styles are available:

| Style | Description |
|-------|-------------|
| **Drawer** | Filters open in a side panel when clicked |
| **Vertical** | Filters displayed as a sidebar alongside the product grid |
| **Horizontal** | Filters displayed in a bar above the product grid |

`[SCREENSHOT: Three filter layout styles side by side]`

---

## Default expanded tabs

For **Drawer** and **Vertical** styles: use the **Tabs always open** field to define which filter tabs are expanded by default. Enter tab names separated by commas.

Example: `Color, Size, Price`

---

## Filter configuration

Volume fully supports native Shopify filters — no coding required. Color swatches configured via **category metafields** display consistently across product pages, collection filters, and product cards.

**Setup guides:**
- [Configure category metafields for color filters](https://help.shopify.com/en/manual/custom-data/metafields/category-metafields/using-category-metafields#step-two)
- [Use metaobjects for advanced filtering](https://help.shopify.com/en/manual/custom-data/metaobjects)

For display settings of color swatches in filters, see [Filter Swatches](../theme-settings/filter-swatches.md).
