# Collection Card

Configure how collection cards appear in sections like Featured Collections, Collection List, and others.

Global settings are in **Theme settings → Collection Card**. Individual sections can override these settings.

`[SCREENSHOT: Collection cards on storefront]`

---

## Display styles

Two styles are available:

| Style | Description |
|-------|-------------|
| **Title below media** | Collection title displayed beneath the image |
| **Title overlay on media** | Collection title overlaid on top of the image |

`[SCREENSHOT: Title below media vs Title overlay side by side]`

> **Note:** When using **Title overlay**, the overlay background color is taken from the **Color scheme** defined in the section settings.

---

## Settings

| Setting | Description |
|---------|-------------|
| **Media ratio** | Aspect ratio for collection card images |
| **Focal point** | Crop anchor for collection images |
| **Center title** | Centers the title within the card |
| **Show product count** | Displays the number of products in the collection |
| **Show arrow** | Adds a directional arrow to the card |
