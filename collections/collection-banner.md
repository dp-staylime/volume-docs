# Collection Banner

The Collection Banner is a standalone section on the collection page, separate from the Product Grid. This separation allows for flexible layout combinations.

`[SCREENSHOT: Collection Banner with image and title above product grid]`

---

## Layout combinations

You can combine the Collection Banner with the Product Grid in several ways:

| Layout | Description |
|--------|-------------|
| **Collection banner + Product grid** | Standard horizontal banner above the product grid |
| **Collection banner vertical + Product grid** | Vertical banner alongside the grid |
| **Collection banner with subcollections + Product grid** | Banner includes subcollection links |
| **Subcollection banners + Product grid** | No main banner — subcollections displayed as cards |
| **Product grid only** | No banner — grid starts at the top of the page |

`[SCREENSHOT: Different layout combinations side by side]`

---

## Subcollections in banner

You can display subcollections as a horizontal slider within the Collection Banner. This requires a dynamic connection via a metafield.

See [Subcollections](subcollections.md) for the full setup guide.
