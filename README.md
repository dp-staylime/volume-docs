# Welcome to Volume Documentation

### What is Volume Shopify theme?

<figure><img src=".gitbook/assets/Group 19.png" alt=""><figcaption></figcaption></figure>

<p align="center"><a href="https://themes.shopify.com/themes/volume/presets/volume" class="button primary">Install Volume from Shopify Theme Store</a></p>

**Volume** is an advanced Shopify theme designed for high-volume stores with rich navigation and deep customization. Built around the Smart Category Menu — a unique top-level navigation system that creates separate home pages for each product category — Volume delivers a feature-rich, conversion-boosting shopping experience with 40+ sections, mobile-first design, and optimized performance.

Volume theme is a product of [Staylime](https://staylime.com/).
