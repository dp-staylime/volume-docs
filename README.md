# Welcome to Volume Documentation

### What is Volume Shopify theme?

<figure><img src=".gitbook/assets/Group 19.png" alt=""><figcaption></figcaption></figure>

<p align="center"><a href="https://themes.shopify.com/themes/volume/presets/volume" class="button primary">Install Volume from Shopify Theme Store</a></p>

**Volume** is an advanced Shopify theme designed for high-volume stores with rich navigation and deep customization. Built around the Smart Category Menu — a unique top-level navigation system that creates separate home pages for each product category — Volume delivers a feature-rich, conversion-boosting shopping experience with 40+ sections, mobile-first design, and optimized performance.

Volume theme is a product of Key Features

* **Smart Category Menu** — unique top-level navigation with separate landing pages per product category
* **40+ sections** — rich library of content and layout sections
* **Up to 21 color schemes** — per-section color customization
* **Advanced typography** — granular heading control with special text styles ({curly} and \[square bracket] effects)
* **Flexible product layouts** — 4 product page layouts with grids, slideshows, and media overlap
* **Color swatches** — native category-based and legacy HEX/image swatches
* **Custom metafields** — subcollections, linked products, size guide, product subtitle, and more
* **Custom options** — 10 types of line item property inputs (v3.0.0+)
* **Store Selector** — built-in multi-location drawer
* **Mobile-first design** — optimized for mobile performance and UX
* **300+ built-in icons** — from Icons8, ready to use in any section

### Quick Start

1. [Install Volume](https://themes.shopify.com/themes/volume/styles/modern) from the Shopify Theme Store
2. Open the **Theme Editor** (Online Store → Themes → Customize)
3. Go to **Theme Settings** to set your colors and typography
4. Configure your **Navigation** structure
5. Browse the **Sections** library to build your pages
