# Filter Swatches

Filter Swatches settings control how color and attribute filters appear in collection sidebars and filter panels, letting customers narrow down products visually.

### Swatch Display

Choose how filter options are rendered:

* **Color swatches** — circular color chips for color-type filters
* **Image swatches** — custom uploaded images as filter options
* **Buttons** — text buttons for non-color attributes (size, material, etc.)

### Swatch Size

Set the diameter of filter swatches:

* **Small** — compact, fits many options in a row
* **Medium** — default, balances visibility and density
* **Large** — prominent swatches, best for fewer color options

### Layout

| Setting             | Description                                                |
| ------------------- | ---------------------------------------------------------- |
| Swatches per row    | Wrap swatches after N items                                |
| Show swatch tooltip | Display filter option name on hover                        |
| Show more threshold | Collapse filter list after N items with a "Show more" link |

### Selected State

Define the visual style for a selected filter swatch:

* **Checkmark** — overlay checkmark icon
* **Border** — highlight border around selected swatch
* **Both** — checkmark and border combined

### Out-of-Stock Swatches

Control how unavailable filter options are displayed:

* **Show** — display normally
* **Cross out** — show with a diagonal line
* **Hide** — remove from the filter panel
