# Custom CSS

<figure><img src="../../.gitbook/assets/Group 18 (1) (1).jpg" alt=""><figcaption></figcaption></figure>

Add custom CSS styles that apply across your entire store.

Paste your CSS directly into the built-in code field. Styles apply globally to all pages.

{% hint style="success" icon="sparkles" %}
**Tip:** Always add custom CSS here rather than editing `base.css` directly — styles in Theme settings → Custom CSS survive theme updates.
{% endhint %}

For guidance on writing CSS for Shopify themes, see [Shopify's documentation](https://help.shopify.com/en/manual/online-store/themes/theme-structure/extend/add-css).
