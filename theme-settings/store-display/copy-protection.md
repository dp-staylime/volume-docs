# Copy protection

<figure><img src="../../.gitbook/assets/Group 18 (1).jpg" alt=""><figcaption></figcaption></figure>

Basic protection against content theft by restricting browser interactions.

| Setting                           | Description                                          |
| --------------------------------- | ---------------------------------------------------- |
| **Disable right click on images** | Prevents visitors from right-clicking to save images |
| **Disable text selection**        | Prevents visitors from selecting and copying text    |

{% hint style="warning" icon="lightbulb" %}
**Note:** These are basic deterrents, not technical protections. Determined users can still access content through browser developer tools.
{% endhint %}
