# Quick search

Configure the search overlay that appears when clicking the search icon.

<figure><img src="../.gitbook/assets/Group 12.jpg" alt=""><figcaption></figcaption></figure>

***

### General

**Enable quick search** — activates the search overlay. Search result types and filters are configured in the [Search & Discovery app](https://apps.shopify.com/search-and-discovery).

***

### Result display

<figure><img src="../.gitbook/assets/Group 14.jpg" alt=""><figcaption></figcaption></figure>

| Setting               | Description             | Products | Collections | Blog posts |
| --------------------- | ----------------------- | -------- | ----------- | ---------- |
| **Layout**            | Grid or List view       | ✓        | ✓           | ✓          |
| **Grid type**         | Number of columns       | ✓        | ✓           | ✓          |
| **Image ratio**       | Aspect ratio for images | ✓        | ✓           | ✓          |
| **Image focal point** | Crop anchor point       | ✓        | ✓           | ✓          |
| **Show vendor**       | Product vendor on cards | ✓        | —           | —          |
| **Show price**        | Product price on cards  | ✓        | —           | —          |
| **Show image**        | Post thumbnail          | —        | —           | ✓          |
| **Show date**         | Publication date        | —        | —           | ✓          |

***

### Popular searches

Displays up to 5 links when the search field is empty — useful for promoting collections or products.

<figure><img src="../.gitbook/assets/Group 12 (1).jpg" alt=""><figcaption></figcaption></figure>

| Setting             | Description                                                   |
| ------------------- | ------------------------------------------------------------- |
| **Menu**            | Select a menu to pull links from. Nested items are not shown. |
| **Show menu title** | Displays the menu name as a heading                           |
| **Link size**       | Typography style for links                                    |
| **Title size**      | Typography style for the section heading                      |

***

### 'All results' button

**Button label** — custom text for the button linking to the full search results page. Leave empty to hide.
