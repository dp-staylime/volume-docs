# Colors & Color Schemes

Volume theme supports up to **21 color schemes**. Each color scheme defines a unique set of colors applied to a section or page element.

### Color Scheme Properties

Each scheme includes the following color slots:

| Property       | Description                                      |
| -------------- | ------------------------------------------------ |
| **Background** | Page or section background color                 |
| **Surface**    | Cards, overlays, secondary backgrounds           |
| **Text**       | Primary text color                               |
| **Primary**    | Buttons, links, interactive elements             |
| **Accent**     | Highlighted elements, badges, decorative details |

### Setting Up Color Schemes

1. In the Theme Editor, go to **Theme settings → Colors**
2. Click **Add scheme** to create a new color scheme
3. Assign colors to each property using the color picker
4. You can create up to **21 schemes**

### Assigning Color Schemes to Sections

Each section in the Theme Editor has a **Color scheme** setting. Select the scheme you want to apply to that section.

This lets you create visual contrast between sections — for example, a dark hero banner followed by a light product grid.
