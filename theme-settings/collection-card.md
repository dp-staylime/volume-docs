# Collection Card

Collection Card settings control how category and collection tiles appear on your homepage, navigation menus, and collection list pages.

### Card Style

Choose the visual presentation for collection tiles:

* **Image with overlay** — collection image fills the card; title appears as text overlay
* **Image above text** — image on top, collection title below
* **Text on image** — centered title directly on the image with optional background

📸 SCREENSHOT: Collection card style options in Theme settings → Collection Card

### Image Settings

| Setting     | Description                                                  |
| ----------- | ------------------------------------------------------------ |
| Image ratio | Aspect ratio — Square (1:1), Portrait (3:4), Landscape (4:3) |
| Image fit   | Cover (crop to fill) or Contain (show full image)            |

### Overlay Settings

When using the **Image with overlay** or **Text on image** style:

* **Overlay color** — background color behind the title
* **Overlay opacity** — transparency of the overlay (0–100%)
* **Text color** — title color on the overlay

### Text Settings

* **Title size** — Small, Medium, or Large
* **Title alignment** — Left, Center, or Right
* **Show product count** — display number of products in collection

### Hover Effect

Choose what happens on hover:

* **None** — static card
* **Zoom** — image scales up
* **Brighten** — image lightens slightly
