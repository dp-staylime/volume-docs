# Product page layout

Configure media display, layout containers, and content structure for the product page from **Theme Editor → Default product → Product overview section**.

`[SCREENSHOT: Product overview section settings in Theme Editor]`

---

## Media style

Two media display styles are available:

**Grids** — stacked or grid arrangement of product images:
- Stack
- Grid
- Alternative grid 1
- Alternative grid 2

**Gallery** — horizontal image slideshow with thumbnails below.

`[SCREENSHOT: Grid style vs Gallery style side by side]`

---

## Media settings

| Setting | Description |
|---------|-------------|
| **Position** | Place media on the left or right side |
| **Size** | Preferred media width from 4/12 to 9/12 |
| **Fit media inside container** | Preserves original image aspect ratio with whitespace around it — useful when product images have different ratios |
| **Container inner spacing** | Padding around media when Fit inside container is enabled |
| **Media gallery color scheme** | Background color for the media container |

---

## Thumbnails

Controls placement, size, shape, and behavior of media thumbnails.

Enable **Sticky thumbnails on desktop** to keep the thumbnail strip fixed while scrolling product content.

> **Note:** Sticky thumbnails are not compatible with Gallery media style.

---

## Overlap

When enabled, the header becomes transparent and overlaps the top of the Product overview section — useful when product images have a uniform background color.

> **Tip:** Add an **Offset** setting equal to the approximate header height to prevent the header from covering product content.

---

## Layout

Controls how content blocks are divided between containers.

| Layout option | Description |
|--------------|-------------|
| **Only first container** | All blocks stacked vertically — classic single-column layout |
| **First and second stretched vertically** | Two containers stacked, each stretching to fill height — recommended for minimal content |
| **Containers in two columns** | Side-by-side containers — each with alignment and padding options |
| **Second container below media** | Essential content above the fold, detailed description placed below media |

`[SCREENSHOT: Four container layout options]`

> **Note:** In two-column layout, media width may be reduced on smaller screens to preserve minimum content width.
