# What's the difference between Adelle, Bloop, Fairmont, Vortex, and Avante?

Adelle, Bloop, Fairmont, Vortex, and Avante are built-in theme styles inside the Avante theme.

They are not separate themes. You do not need to buy them individually.

Switching styles changes the overall visual look of your store, including colors, fonts, and other design settings.

### Switch between theme styles

1. Open **Theme Editor**.
2. Click **Theme settings** on the left.
3. Open **Theme style**.
4. Select the preset you want to use.

{% hint style="warning" icon="triangle-exclamation" %}
Switching presets updates visual settings only.

Shopify does not automatically recreate the full demo content structure.

If you want the exact layout from one of the demos, install that preset fresh from the Shopify Theme Store.
{% endhint %}
