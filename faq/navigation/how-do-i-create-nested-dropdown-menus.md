# How do I create nested / dropdown menus?

Menu nesting is configured in **Shopify Admin**, not in the Theme Editor.

### Steps

1. Go to **Shopify Admin → Content → Menus**
2. Open the menu you want to edit
3. Drag a menu item slightly to the **right** under a parent item to make it a child link

<figure><img src="../../.gitbook/assets/Screenshot 2026-06-11 at 14.44.02.png" alt=""><figcaption></figcaption></figure>

4. Save the menu

The theme automatically renders the nested structure as dropdowns or a Mega menu, depending on your Header settings.

{% hint style="warning" icon="lightbulb" %}
Avante supports up to **3 levels** of menu nesting.
{% endhint %}
