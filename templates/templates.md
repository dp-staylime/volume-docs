# Overview

Templates define the layout of each page type in your store. Each template is a collection of sections and blocks configured independently in the Theme Editor.

Changes to a template apply to **all pages using that template**. To customize a page differently from others of the same type, assign it to a separate template.

***

## Available templates in Avante

| Template               | Used for                                   |
| ---------------------- | ------------------------------------------ |
| **Default product**    | All products by default                    |
| **Pre-order**          | Products available for pre-order           |
| **Default collection** | All collection pages                       |
| **Collections list**   | The `/collections` page                    |
| **Default blog**       | Blog listing page                          |
| **Default blog post**  | Individual blog articles                   |
| **Cart**               | The `/cart` page                           |
| **Search**             | The `/search` page                         |
| **Page**               | Standard pages (About, Contact, FAQ, etc.) |
| **Password**           | Store password page                        |
| **404**                | Page not found                             |
| **Gift card**          | Gift card confirmation page                |
| **Quick view**         | Product quick view popup                   |

***

## How to assign a template to a page or product

**For products:** Go to **Shopify Admin → Products** → open a product → find **Theme template** on the right → select a template → Save.

<figure><img src="../.gitbook/assets/Screenshot 2026-06-17 at 17.10.21.png" alt=""><figcaption></figcaption></figure>

**For pages:** Go to **Shopify Admin → Online Store → Pages** → open a page → find **Theme template** on the right → select a template → Save.

<figure><img src="../.gitbook/assets/Screenshot 2026-06-17 at 17.11.16.png" alt=""><figcaption></figcaption></figure>

{% hint style="warning" icon="triangle-exclamation" %}
A template only appears in the dropdown if it exists in your **currently published live theme**. Templates created in a \[DEV] copy won't show here — create them in your live theme first.
{% endhint %}
