# Mobile header

Avante has a separate mobile header configuration for screens below the desktop breakpoint.

## Setting up

In **Theme Editor**, switch the preview to mobile using the device toggle at the top. The **Mobile header** section controls what appears on small screens:

* Logo
* Menu trigger (hamburger icon)
* Search
* Account
* Cart

## Hamburger icon style

Go to **Theme settings → Navigation → Burger menu** and choose between icon styles: Classic, Classic wide, Two stripes, Two stripes wide, Alternate.

## Menu drawer on mobile

When the burger icon is tapped, the **Menu drawer** opens. Configure its layout in **Theme settings → Navigation → Menu drawer style**: Standard or Mega menu drawer (v7).
