# Header

The header is the first touchpoint for your visitors. Avante supports two primary navigation layouts — Header-based and Sidebar-based — configured in **Theme settings → Appearance → Navigation layout**.

## Setting up the header

1. Go to **Theme Editor → Header section**
2. Add and arrange blocks: Logo, Navigation, Search, Account, Cart
3. Set **Block horizontal alignment**: Stretched, Packed Left/Center/Right, or Space between
4. Each block has its own **Element alignment** setting for fine-tuning

## Adding a logo

Go to **Theme settings → Logo** and upload your image. If you're using a transparent header, also upload a separate version under **Logo → Image for transparent navigation** — this version displays over dark backgrounds.

## Navigation

Select your main menu under **Header → Menu**. To configure what appears in the menu, go to **Shopify Admin → Online Store → Navigation**.

## Mega menu

Add a **Mega menu** block inside the Header section. In the **Dropdown link** field, enter the exact name of the parent menu item as it appears in Shopify Admin (e.g. `Collections`).

> ⚠️ Do not translate the Dropdown link field using Translate & Adapt — this breaks the connection to your navigation tree.

## Hover mega menu (v12.2)

Add a **Hover mega menu** block to enable cursor-triggered dropdowns on desktop. Configure it the same way as the regular Mega menu. See the Vortex preset for a live example.

## Transparent header

See [Transparent header](../features-and-advanced-setup/transparent-header.md) for full setup instructions.

## Sticky header

Enable **Sticky behavior** in the Header section settings. Options: None, Always, On scroll up.
