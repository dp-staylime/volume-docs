# Footer

The footer appears on every page. Configure it in **Theme Editor → Footer section**.

## Adding blocks

Click **Add block** inside the Footer section to add:

* Menu columns
* Newsletter signup
* Social links
* Localization (language/currency selector)
* Store Selector (v11)
* Text / Custom HTML
* Back to top button

## Layout

Use the **Columns** setting to control how many columns the footer uses on desktop. On mobile, blocks stack vertically.

## Social links

Add your social media URLs in **Theme settings → Social media**. The footer Social links block will automatically display the icons for the networks you've filled in.

## Newsletter

The Newsletter block connects to your Shopify email subscribers. No third-party app required — subscribers are added to your **Shopify Admin → Customers** list.
