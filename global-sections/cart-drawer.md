# Cart drawer

The cart drawer slides in from the right when a customer adds a product or clicks the cart icon.

## Enabling features

Go to **Theme settings → Cart drawer** to configure:

| Setting              | What it does                                                     |
| -------------------- | ---------------------------------------------------------------- |
| Discount code input  | Lets customers apply discount codes directly in the drawer (v12) |
| Order notes          | Adds a text field for delivery instructions                      |
| Cart recommendations | Shows recommended products below the cart items                  |
| Free shipping bar    | Shows progress toward free shipping threshold                    |

## Free shipping bar

Enable in **Theme settings → Cart** and set the threshold amount in your store's primary currency. For different thresholds per market, configure the value via **Shopify Translate & Adapt → Default theme content → Cart → Free Shipping Minimum**.

## Cart drawer vs Cart page

Both exist independently. The Cart drawer is the slide-out panel; the Cart page is a standalone `/cart` URL. Configure them separately under **Theme settings → Cart drawer** and **Theme settings → Cart**.
