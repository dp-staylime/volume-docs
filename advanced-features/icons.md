# Icons

Volume includes a built-in library of over **300 icons** that can be used across sections, blocks, and buttons to enhance visual content.

`[SCREENSHOT: Icon library preview]`

---

## Where to use icons

Icons can be added to the following blocks and sections:

- Scrolling promotion
- Icons with text
- Advantages
- Featured collection with media
- Buttons (before or after text)

---

## How to add an icon

1. Open the icon library: [https://volume-theme-fashion.myshopify.com/pages/icons](https://volume-theme-fashion.myshopify.com/pages/icons)
2. Find the icon you want to use
3. Click the icon — its URL is automatically copied
4. Paste the URL into the **Icon** field of the relevant block, section, or button

`[SCREENSHOT: Pasting icon URL into the Icon field in Theme Editor]`

---

## Customization

| Setting | Description |
|---------|-------------|
| **Icon size** | Set independently for desktop and mobile |
| **Icon color** | Custom color per icon |
| **Swap icon position** | In buttons: place the icon before or after the text |

> **Note:** For two-tone icons, the line color (stroke) is inherited from the **Text** color of the section's color scheme.
